### GraphCMS – Headless CMS Blog Application  
**GraphCMS** is a **blog application** inspired by **Medium**, built to demonstrate seamless integration of **GraphQL** with modern frontend technologies.  
Powered by **Hygraph (Headless CMS)** and **Next.js**, this application provides a dynamic and scalable content management solution.  

By leveraging **GraphQL**, GraphCMS enables efficient querying and management of content, resulting in a fast, responsive, and interactive blogging experience.  
With **Tailwind CSS** for styling and **Next.js static site generation (SSG)**, the platform ensures optimal performance and SEO benefits.

- **📝 Dynamic Blog Content** – Managed through **Hygraph (Headless CMS)** with GraphQL-powered data queries.
