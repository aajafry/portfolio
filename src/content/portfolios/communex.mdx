### CommuneX – Real-time Chat Application  
**CommuneX** is a fully responsive, real-time chat application built using the **MERN stack** and **Socket.IO** for seamless and dynamic communication.  
It features secure user authentication, live messaging, and supports group conversations. CommuneX is optimized for scalability, making it ideal for small to large-scale environments.

#### 🎯 Key Features  
- **💬 Real-time Messaging** – Instant, live chat powered by **WebSockets** for fluid communication.  
- **🔒 Secure Authentication** – **JWT (JSON Web Tokens)** and cookie-based secure login system.  
- **👥 Group Chat Support** – Effortlessly create and manage group conversations.  
- **📱 Responsive Design** – **Tailwind CSS** ensures optimal performance across mobile, tablet, and desktop.  
- **⚡ Scalable Architecture** – Built with **Node.js, Express.js, and MongoDB** for performance and scalability.
