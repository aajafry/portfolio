### Recto – React Dashboard Template  
**Recto** is a modern **React-based dashboard template** designed to explore **data visualization** and **event management** through **Chart.js** and **FullCalendar.js**.   
It implements **secure token-based authentication** with **Passport.js (oAuth)**, offering a seamless experience for user login and data management.

#### 🎯 Key Features  
- **📊 Interactive Data Visualization** – Built with **Chart.js** for dynamic, responsive charts and graphs.  
- **📅 Calendar & Event Management** – FullCalendar.js integration for seamless scheduling and event tracking.  
- **🔐 Secure Authentication** – oAuth 2.0-based authentication using **Passport.js** and **JWT** for secure access.  
- **🧱 Modular Design** – Follows **atomic design principles** for enhanced scalability and reusability.  
- **🛠️ SOLID Architecture** – Promotes best practices with **SOLID principles** to ensure robust and maintainable code.  
- **📱 Fully Responsive** – Optimized across mobile, tablet, and desktop environments.  
- **⚙️ Custom Hooks** – Implements reusable hooks to manage complex logic efficiently.
